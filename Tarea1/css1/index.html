Bienvenidos a mi Sitio

Un diseño sencillo, limpio y estructurado con CSS fundamental.

Este proyecto demuestra el uso de propiedades básicas de CSS como márgenes, rellenos y bordes para organizar la información sin depender de frameworks externos.

Mis Objetivos:

  • Aprender la base del modelo de caja (Box Model).
  • Dominar el posicionamiento estático y de bloque.
  • Entender la jerarquía visual mediante tipografía.
  • Controlar las dimensiones fijas.
